# MonoCard

Crypto-powered card platform built on Monolythium.

## What We're Building

MonoCard brings cryptocurrency spending to everyday life with virtual and physical debit cards backed by your crypto holdings.

### Repositories

| Repo | Description |
|------|-------------|
| [mono-card-backend](https://github.com/monocard/mono-card-backend) | REST API powering the card platform |
| [mono-card-web](https://github.com/monocard/mono-card-web) | Web dashboard for card management |
| [mono-card-mobile](https://github.com/monocard/mono-card-mobile) | iOS and Android mobile app |
| [mono-card-browser](https://github.com/monocard/mono-card-browser) | Browser extension for quick access |
| [mono-card-app-design](https://github.com/monocard/mono-card-app-design) | Design system and tokens |
| [mono-card-website](https://github.com/monocard/mono-card-website) | Marketing website |
